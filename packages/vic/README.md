# Victorinix

Friendly and useful box of portable computer tools + provision, packaging, distributing and install stuff. Powered by [Nix](https://nixos.org) and [MiZe](https://mize.work).

[About](https://wiki.ppc.social/Victorinix)

The name Victorinix comes from Victorinox, the company that makes the famous Swiss Army Knives.
